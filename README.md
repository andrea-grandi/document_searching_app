# Apache Solr search engine

## Setup & Installation

Make sure you have the latest version of Python installed.

```bash
git clone <repo-url>
```

```bash
pip install -r requirements.txt
```

## Running The App 

For MacOS 

```bash 
python macOS_app.py
```

For Windows

```bash
python windows_app.py
```

## Viewing The App

Go to `http://127.0.0.1:5000`
